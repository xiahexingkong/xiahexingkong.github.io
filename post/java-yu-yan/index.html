JAVA语言(1)

2022-01-27 JAVA

⭐️for循环，boolean类型

🏳️‍🌈for循环
例：输出100以内的素数。

for(int n=2;n<100;n++){
    int flag=0;
    for(int i=2;i*i<=n;i++){
        if(n%i==0){
            flag=1; //当判断n为质数，标记
            break;  //跳出当前循环判断下一个n
        }
    }
    if(flag==1){    //根据判断n是否被标记来输出
    }
    else{
        System.out.println(n+"是素数");
    }
}

例：输出前50个素数。

int cout=0;              //计数标志
for(int n=2;cout<50;n++){//[0~50)个计数循环共50个
    int flag=0;
    for(int i=2;i*i<=n;i++){
        if(n%i==0){
            flag=1;      //当判断n为质数，标记
            break;       //跳出当前循环判断下一个n
        }
    }
    if(flag==1){         //根据判断n是否被标记来输出
    }
    else{
        cout=cout+1;
        System.out.println("第"+cout+"素数是："+n);
    }
}

例：boolean a=ture;//布尔类型定义

① !a (逻辑非)

③a||b (逻辑或)

答：①x>4&&x<6 ②x>=4&&x<=6
