🏴例：输入指定个数的整数，求出平均数，并将大于平均数的值输出。

Scanner in =new Scanner(System.in);
int x=in.nextInt();
double sum=0.0,avage;
int [] numbers=new int [x];
for(int i=0;i<numbers.length;i++){//对数组的遍历，提高程序的可扩展性
    int n=in.nextInt();
    numbers[i]=n;
    sum+=numbers[i];
}
avage=sum/x;
for(int i=0;i<numbers.length;i++){
    if(numbers[i]>=avage){
        System.out.println(numbers[i]);
    }
}
System.out.println(avage);

🔶数组的遍历：

int a[]={12,23,34,45};
for(int k: a){//foreach循环:每次从a中依次取一个数据传给k
    System.out.println(k);
}

🔶素数遍历判断：
例:输出100以内的素数。

boolean []numbers=new boolean[100];
for(int i=0;i<numbers.length;i++){
    numbers[i]=true;
}
for(int i=2;i<numbers.length;i++){
    if(numbers[i]==true)
        for(int j=2;j*i<numbers.length;j++){
            numbers[i*j]=false;
        }
}
for(int i=2;i<numbers.length;i++){
    if(numbers[i]==true){
        System.out.println(i);
    }
}

例:输出50个素数：

int []numbers=new int[50];
numbers[0]=2;
int cnt=1;
OUT_for:
for(int i=3;cnt<50;i++){    //限制素数的个数
    for(int j=0;j<cnt;j++){
        if(i%numbers[j]==0){
            continue OUT_for;
        }
    }
    numbers[cnt++]=i;
}
for(int k:numbers){
    System.out.println(k);
}
